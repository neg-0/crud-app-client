# Initial set up

## Clone the repo

```
git clone git@github.com:neg-0/crud-app-client.git
```

## Create the .env file in the root directory

```
touch .env
```

## Add the following to the .env file

```
VITE_API_URL=http://localhost:3000
```

## Install dependencies

```
npm install
```

## Run the app

```
npm start dev
```
